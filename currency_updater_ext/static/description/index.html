Currency rate update extension

Note that this module works only with National Bank of Poland service!

This module depends on 'Currency rate update' module. It adds the 'Table name' and 'Table rate date' column.

Configuration

1.The base currency must be PLN. (pic 1)

2.And its base rate must be 1. (pic 1)

3.Remember also that this module works only if you choose the National Bank of Poland. (pic 2)
