What's inside ?

This Odoo module increases length of meta Title and meta Description fields, brings better Google preview and adds tips. Putting more text in meta tags help Google to present good search results and to attract visitors.

SEO meta

Show more on search engines results page

More words in search engine's index

Odoo limits meta Title to 70 characters, but Google doesn't have such limit. The two real limits are :

  1. for the home page, the site name must be in the first 600 pixels (nearly 70 characters), for the other pages the site name must end each meta Title ;
  2. on the search results page, Google will display only 70 characters (Bing 40).
But meta Title length is not otherwise limited and search engines will parse the entire title for their index. This module allows 120 characters in meta Title.

Decide for Google

Google prints nearly 160 characters for each search results. It is often a sentence from the Description field. But if there is no search related sentence in your Description, Google will pick words from the page content. Help Google ! write longer descriptions, more sentences, more words. Google will choose one sentence from the available ones. This module allows 320 characters in meta Description. Note that meta Description is not related to SEO, but to Search Engines Result Page (SERP). Google will not use it for referencing your page but to print its search results.
