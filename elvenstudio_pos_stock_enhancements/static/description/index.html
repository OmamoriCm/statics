POS stock enhancements

  • Manage auto transfer picking
  • Set your custom return picking type
This module improve your POS experience

Installation & Settings

To install this module, you need to follow this steps:

  • download the module and put it into your Odoo addons folder;
  • update the modules list by clicking on the "Update Modules List" link under "Settings -> Modules";
  • search and install the module by clicking on the install button.

How to use this module

It's possible enable auto trasfer picking from 'Poin of Sale / Configuration / Point of Sale'.
Select in the Tree View your point of sale.


Activate the checkbox 'Auto Transfer Picking'.


After a sale of a product in POS, go on the Tree View of orders ('Point of Sale / Orders') and open the order sold previously. Under the tab 'Extra Info', there is the related picking.


If you click on the related picking link, you will see the screen of transfer done with the correct destination location.

You can setting custom return picking type, from 'Point of Sale / Point of Sales', choose your Poin of Sale, and select correct option in 'Return pickin type' field.
